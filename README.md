# sql-employee-database-analysis
SQL project exploring and analysing structured data using MySQL, with queries designed to investigate patterns and relationships within the dataset.

## Project Structure

- `employee_database_pt1.sql`  
  Database setup and initial exploration of employee data, including table creation and basic queries.

- `employee_database_pt2.sql`  
  Analytical SQL queries examining employee salaries, demographics, and patterns within the dataset.
